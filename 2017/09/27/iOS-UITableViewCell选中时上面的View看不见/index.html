iOS-UITableViewCell选中时上面的View看不见

现象

cell被选中时，cell上面的view全部都看不见，即时有给view设置颜色

官方文档

UITableViewCell changes the background color of all sub views when cell is selected or highlighted.
意思就是说当UITableViewCell被选中或者高亮的时候，它的所有子view的颜色都会改变。

解决方法

重写UITableViewCell的两个方法

- (void) setHighlighted:(BOOL)highlighted animated:(BOOL)animated {
    [super setHighlighted:highlighted animated:animated];

    self.testview.backgroundColor = [UIColor redColor]; // 你要保持不变的view
}

- (void) setSelected:(BOOL)selected animated:(BOOL)animated {
    [super setSelected:selected animated:animated];

    self.testview.backgroundColor = [UIColor redColor]; // 你要保持不变的view
}
